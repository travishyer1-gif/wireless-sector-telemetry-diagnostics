# X Post Draft

Built a sanitized public case study from a real wireless ISP field diagnostic.

The tool compares sector telemetry across customer radios to spot AP-side receive asymmetry, retry/CCQ issues, and before/after health changes. The real win was narrowing a messy sector problem into a field action that largely resolved it.
